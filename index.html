TowerShot Support

Help with turning overlapping screenshots or a screen recording into one seamless long image.

Quick tips

  1. Select screenshots in top-to-bottom order and keep some repeated content between each image.
  2. Keep the page width and zoom unchanged while capturing.
  3. For screen recordings, lock orientation and scroll slowly downward.
  4. If a join needs help, use Fine Tune to move either screenshot directly.

Screen recording not detected?

Wait for the recording to finish saving in Photos, return to TowerShot, then choose Use a Screen Recording again. Full Photos access is needed for automatic discovery of the new recording.
